# tax-plus-node

# Usage
## nodemon
```bash
git clone git@github.com:solareenlo/tax-plus-node.git
cd tax-plus-node
docker-compose up -d --build
```
で, nodemonが走って, `localhost:3002/api/v1`にアクセスする.

## lint
```bash
git clone git@github.com:solareenlo/tax-plus-node.git
cd tax-plus-node
npm i
npm run lint
```

## build
```bash
git clone git@github.com:solareenlo/tax-plus-node.git
cd tax-plus-node
npm i
npm run build && start
```
